# 💼 Employee Payroll Management System 

## 📌 Project Overview
The Employee Payroll Management System is a SQL-based mini project designed to manage employee records, department details, and salary information.

This project demonstrates strong understanding of relational databases, SQL queries, joins, subqueries, views, indexing, and stored procedures.

---

## 🎯 Objectives
- Manage employee and department data
- Calculate total salary (Basic + Bonus - Deduction)
- Find highest and second highest salary
- Perform department-wise salary analysis
- Implement views and stored procedures

---

## 🗂️ Database Schema

### 1️⃣ Department Table
- department_id (Primary Key)
- department_name

### 2️⃣ Employee Table
- emp_id (Primary Key)
- name
- age
- gender
- department_id (Foreign Key)
- joining_date

### 3️⃣ Salary Table
- emp_id (Foreign Key)
- basic_salary
- bonus
- deduction

---
